# Python Basics for Beginners

This repository contains beginner friendly Python programs with simple explanations.

I created this project while learning Python, Git, and GitHub during my internship preparation.

The programs are written in an easy and understandable way so that beginners can learn step by step.




## Topics Covered

- Introduction to Python
- Variables and Data Types
- Conditionals and Loops
- Functions
- Exception Handling
- Object Oriented Programming


## What You Will Learn

- How to write Python programs
- How variables work
- How decision making works using if else
- How loops repeat code
- How to create and use functions
- How to handle errors using exception handling
- Basics of Object Oriented Programmin


## Goal of This Repository

The goal of this repository is to practice Python basics and improve programming skills by building simple beginner friendly examples.


## How to Run

1. Install Python
2. Clone this repository
